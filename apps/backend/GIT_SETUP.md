# Git Repository Setup - PMD Management System

## Prerequisites

Ensure Git is installed on your system:
- **Windows:** Download from https://git-scm.com/download/win
- **Mac:** `brew install git` or download from https://git-scm.com/download/mac
- **Linux:** `sudo apt-get install git` or `sudo yum install git`

## Initial Setup Commands

Run these commands in your project root directory:

```bash
# Initialize git repository
git init

# Add all files to staging
git add .

# Create initial commit
git commit -m "Initial PMD System Backend"

# Rename branch to main (if needed)
git branch -M main

# Add remote repository (REPLACE TU-USUARIO with your GitHub username)
git remote add origin https://github.com/TU-USUARIO/pmd-system.git

# Push to remote repository
git push -u origin main
```

## Important Notes

### 1. Replace GitHub Username
Before running the `git remote add origin` command, replace `TU-USUARIO` with your actual GitHub username.

**Example:**
```bash
git remote add origin https://github.com/johndoe/pmd-system.git
```

### 2. Create GitHub Repository First
Before pushing, ensure you have created the repository on GitHub:
1. Go to https://github.com/new
2. Repository name: `pmd-system`
3. Description: "PMD Management System Backend"
4. Choose Public or Private
5. **DO NOT** initialize with README, .gitignore, or license (we already have these)
6. Click "Create repository"

### 3. Authentication
If you haven't set up Git credentials, you may need to:
- Use a Personal Access Token (recommended for HTTPS)
- Or set up SSH keys for SSH URLs

**Personal Access Token:**
1. Go to GitHub Settings → Developer settings → Personal access tokens → Tokens (classic)
2. Generate new token with `repo` scope
3. Use token as password when pushing

## Alternative: Using SSH

If you prefer SSH (recommended for security):

```bash
# Add remote with SSH URL
git remote add origin git@github.com:TU-USUARIO/pmd-system.git

# Push to remote
git push -u origin main
```

## Verification

After pushing, verify the repository:

```bash

# View commit history
git log --oneline
```

## Troubleshooting

### Error: "remote origin already exists"
```bash
# Remove existing remote
git remote remove origin

# Add new remote
git remote add origin https://github.com/TU-USUARIO/pmd-system.git
```

### Error: "Authentication failed"
- Use Personal Access Token instead of password
- Or set up SSH keys

### Error: "Repository not found"
- Verify repository exists on GitHub
- Check repository name and username
- Ensure you have access permissions
